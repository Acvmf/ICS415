This is my secondary page. It has some formatting differences in the Bootstrap grid.
Content
The main difference on this page is in the Navigation bar. Each button is a different 1-wide column, as opposed to one column that's 3 wide.
This causes the buttons to never jump to the next line and displace the content area, even when they cross size boundaries. However, when you expand and contract the window, the buttons start overlapping each other or spacing out strangely. I haven't yet decided whether this is preferable to the default.
